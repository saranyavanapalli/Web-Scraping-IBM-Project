# Web-Scraping-IBM-Project
Stock Price vs Revenue Dashboard
This project aims to analyze and visualize the stock prices of Tesla and GameStop alongside their respective revenue data. It provides a comparative analysis of how the stock prices of these companies correlate with their revenue over time.

Introduction
As a data scientist working for an investment firm, understanding the relationship between a company's stock performance and its revenue is crucial. This project extracts historical stock price data using the Yahoo Finance API and fictional revenue data for demonstration purposes.

Features
Extracts historical stock price data for Tesla and GameStop using the Yahoo Finance API.
Generates fictional monthly revenue data for Tesla and GameStop for demonstration purposes.
Creates interactive plots to visualize the stock prices and revenue data.
Compares the stock prices and revenue of Tesla and GameStop over time.
